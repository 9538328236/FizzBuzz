# FizzBuzzAPI
This project is a sample ASP.NET 7 Web API that demonstrates the application of SOLID principles, dependency injection, and the factory design pattern. It also includes unit tests using NUnit and Moq.

# Prerequisites
-- .NET 7 SDK
-- Visual Studio 2022
